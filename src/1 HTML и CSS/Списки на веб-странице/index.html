Определения из frontend-разработки

DOM
Объектная модель документа, способ представления HTML-тэгов в виде вложенных структур, которая позволяет взаимодействовать с каждым элементом посредством селекторов
Favicon
Фавикон, иконка сайта, обычно изображение 16х16 пикселей в формате .ico
Framework
Фреймворк, платформа, определяющая структуру программы, стиль написания, принцип работы программы. Например: React, Angular
Keyframes
Ключевые кадры, используемые для создания CSS-анимаций. Обозначаются как @keyframes
Progressive Web Application (PWA)
Прогрессивное веб-приложение, технология, функционально и визуально трансформирующая сайт в приложение
Preprocessor
Препроцессор, инструмент, преобразующий один синтаксис в другой, предобработка, например, SCSS/SASS
Single Page Application (SPA)
Одностраничное приложение, взаимодействующее с пользователем за счёт динамической перерисовки данных на странице
Web Socket
Протокол прикладного уровня, обеспечивающий постоянное соединение с сервером и возможность получения ответа без отправки HTTP-запроса
